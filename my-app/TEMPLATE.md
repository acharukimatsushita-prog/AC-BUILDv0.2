# my-app template

Use this app as the reference template when starting the next application.

Included stack:

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- shadcn/ui setup with Button and Card
- lucide-react
- TanStack Table
- OpenAI API route
- PostgreSQL connection helper and health route
- Docker and Docker Compose
- ESLint

Useful commands:

```powershell
npm.cmd run dev
npm.cmd run build
npm.cmd run lint
docker compose up --build
```
